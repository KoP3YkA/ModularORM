Modular ORM Documentation

Getting Started

Step 1: Install Dependencies

Before you begin using ModularORM, you need to install the necessary dependencies. Run the following command in your project directory:

npm install typescript ts-node reflect-metadata mysql2

Step 2: Clone the Repository

Next, clone the repository into your project directory. Create a folder named database and clone the repository there:

git clone https://github.com/your-repo/modularorm.git database
