Modular ORM Documentation

Getting Started

Step 1: Install Dependencies

Before you begin using ModularORM, you need to install the necessary dependencies. Run the following command in your project directory:

npm install typescript ts-node reflect-metadata mysql2

Step 2: Clone the Repository

Next, clone the repository into your project directory. Create a folder named database and clone the repository there:

git clone https://github.com/KoP3YkA/ModularORM.git

Step 3: Configure TypeScript

Create a tsconfig.json file in your project and make sure the following options are set:


{
    "compilerOptions": {
        "target": "es2016",
        "module": "commonjs",
        "esModuleInterop": true,
        "forceConsistentCasingInFileNames": true,
        "strict": true,
        "skipLibCheck": true,
        "experimentalDecorators": true,
        "emitDecoratorMetadata": true
    }
}

Step 4: Initialize the Database

To start using ModularORM, import the main class, get its instance, and call the asynchronous start method to connect to your database. Pass your database configuration as follows:


import { ModularORM } from "./database/ModularORM/modularorm";

new Promise(async (re, rj) => {
    const database = ModularORM.getInstance;
    await database.start({
        host: 'localhost',
        password: '',
        user: 'root',
        database: 'orm',
        port: 3306
    })
})

Creating tables

To create a new table, you need to create a class that inherits from the Module class. Then, override the static table() method to return the name of your table. Add the @Table annotation to the class. To create columns, define the class fields, where the field name will correspond to the column name in the database, and add the @Column annotation, passing the necessary parameters.


@Table
class Users extends Module {
    @Column({
        type: ColumnType.INTEGER,
        autoIncrement: true,
    })
    public id : number = 1;

    @Column({
        type: ColumnType.VARCHAR(255),
        notNull: true,
    })
    public userId : string = "";

    public static override get table() {return "users";}
}

IMPORTANT: For tables to be created correctly, the classes must be defined before connecting to the database, that is, before calling the start() method in the main class. If the classes were created later, you need to call the init() method on the Database object (e.g., await Database.init()).

Creating SQL query`s

Database interactions are performed through the QueryBuilder class. You need to create an instance of this class and then use its methods to build the SQL query.


class Chats extends Module {
        @Column({
            type: ColumnType.VARCHAR(255)
        })
        public chatId : string = ''

        public static override get table() {return "chats";}
}

const builder : QueryBuilder = new QueryBuilder()
      .setType(QueryType.UPDATE)
      .setTable(Chats)
      .setWhere(
        new WhereBuilder()
          .equalAnd('chatId', '123')
      )
      .setUpdate(
        new UpdateBuilder()
          .add('chatId', '321')
      )

const query : FinalQuery = builder.build()

Final Query

execute() method: Sends the query to the database and returns nothing.


class Books extends Module {
        @Column({
            type: ColumnType.INTEGER,
            autoIncrement: true
        })
        public bookId : number = 1

        @Column({
            type: ColumnType.INTEGER
        })
        public price : number = 1

        public static override get table() {return "books";}
}

const builder : QueryBuilder = new QueryBuilder()
      .setType(QueryType.INSERT)
      .setTable(Books)
      .setInsert(
            new InsertBuilder()
                .add('price', 3400)
      )
const query : FinalQuery = builder.build()
await query.execute() // INSERT INTO books (price) VALUES (3400)

get(T) method: Sends the query to the database and returns the results as an array of T[] objects, where T is a class that extends QueryResult.


class Books extends Module {
        @Column({
            type: ColumnType.INTEGER,
            autoIncrement: true
        })
        public bookId : number = 1

        @Column({
            type: ColumnType.INTEGER
        })
        public price : number = 1

        public static override get table() {return "books";}
}

class QueryRes extends QueryResult {

    @Result('bookId')
    public id: number = 1;

    @Result('price')
    public price: number = 1;

}

const builder : QueryBuilder = new QueryBuilder()
      .setType(QueryType.SELECT)
      .setTable(Books)
      .setSelect(
            new SelectBuilder()
                .addColumn('bookId')
                .addColumn('price')
      )
const query : FinalQuery = builder.build()
const res : QueryRes[] = await query.get<QueryRes>(QueryRes)
